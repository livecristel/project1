How Its Work

How does it help to keep SAFE AND SECURE?

Homes, offices, and shop have various entry points such as doors, windows, shutters, grills, and glass windows. To secure all possible entry points, wired or wireless sensors such as magnetics contact sensors for doors and sliding windows, vibration detectors for glass doors, glass windows, and grills, shutter contact sensors for shutters and parking gates, PIR motion sensors for open areas, and FIR/SMOKE/LPG Gas sensors to detect fire and gas leakages will be installed. An emergency panic switch will be available.

Different types of Intrusion Detection

UTPL makes different types of intrusion Detection and Alarm System to fit the needs of each individual user. It doesn't matter if you own a home, small business, bank, ATM, Industry, or retail/office, we have a system that's right for you. We offer different products for you to choose from.


WIRED SECURITY SYSTEM

A wired security system is a home or business security system that uses physical wires to transmit signals between the various sensors and panel

WIRELESS SECURITY SYSTEM

A wireless security system is a home or business security system that uses wireless technology to transmit signals between sensor and panel.

HYBRID SECURITY SYSTEM

A hybrid security system is a home or business security system that combines both wired and wireless technology to transmit signals between sensors and panel, it offers the advantages of both wired and wireless system and easy to install and maintain.

TELIMETRY CONECTIONS

The panel has various telemetry connection options like CSM call based, SMS based or internet based. the panel has additional connection options of Wi-Fi, blue tooth and RF ID.

MOBILE APP

A mobile app security system is a type of home or business security system that can be controlled and monitored using a mobile app. The system has Different level of app for notification, control and configure, LOT configurations etc.
